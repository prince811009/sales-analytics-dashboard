# Operations Analytics Dashboard

A Flutter-based operations analytics dashboard that transforms CSV data into KPI dashboards, data exploration, forecasting insights, and management reports.

## Current Features

- CSV data upload
- KPI dashboard
- Sales trend chart
- Data explorer
- Management report
- Forecast summary

## Tech Stack

- Flutter
- Dart
- CSV parsing
- fl_chart
- file_picker
